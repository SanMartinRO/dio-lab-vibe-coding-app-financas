# 💸 App de Organização de Finanças Pessoais do San com Vibe Coding

PRD refinado no Copilot Web:

```markdown
PRD – Aplicativo de Organização de Finanças Pessoais

1. Contexto
O objetivo é criar um aplicativo de organização de finanças pessoais que funcione por meio de conversas em linguagem natural.
A proposta é simplificar o controle financeiro, eliminando a necessidade de formulários complexos ou planilhas manuais, tornando a experiência mais acessível e intuitiva.

2. Problema
Atualmente, muitos usuários desistem de controlar seus gastos porque:
- Os aplicativos exigem entrada manual excessiva.
- Há pouca personalização na experiência.

A solução proposta é oferecer uma interface conversacional, com recomendações automáticas de economia, para reduzir fricção e aumentar engajamento.

3. Público-Alvo
- Pessoas que desejam começar a organizar suas finanças de forma prática e sem complicação.
- Principalmente iniciantes que não têm familiaridade com planilhas ou ferramentas financeiras tradicionais.
- Usuários diversos, com diferentes níveis de experiência digital, garantindo acessibilidade por meio de Design Universal.

4. Funcionalidades-Chave
1. Registro de gastos via chat em linguagem natural.
2. Classificação automática das transações (alimentação, transporte, lazer etc.).
3. Definição e acompanhamento de metas financeiras (ex.: poupar R$200/mês).
4. Agente Financeiro que fornece dicas personalizadas de economia.
5. Relatórios simples e personalizados, com visualização clara de gastos e progresso.
6. Design Universal: interface inclusiva, acessível e adaptada para diferentes perfis de usuários (ex.: suporte a leitores de tela, linguagem clara, contraste adequado, navegação intuitiva).

5. Entregável da IA
- Plano de MVP contendo:
  - Principais telas (chat, metas, relatórios).
  - Recursos necessários (NLP, categorização automática, motor de recomendações).
  - Esboço de validação inicial (testes com usuários iniciantes e diversos perfis).
- Linguagem acessível e tom educativo, em português.
- Garantia de Design Universal como princípio orientador.
```

# Funcionalidades do Aplicativo de Finanças Pessoais

## 1. Registro de Gastos via Chat
- Permite registrar despesas e receitas em **linguagem natural**.
- Exemplos: "Gastei 50 reais no almoço", "Recebi 4500 de salário".
- O assistente interpreta automaticamente e atualiza os registros.

## 2. Classificação Automática de Transações
- As despesas são **categorizadas automaticamente** (moradia, alimentação, saúde, lazer, transporte etc.).
- Facilita a visualização dos gastos sem necessidade de organização manual.

## 3. Metas Financeiras
- Usuário pode definir **objetivos financeiros** (ex.: reserva de emergência, viagem, compra de notebook).
- Cada meta mostra:
  - Progresso
  - Valor acumulado
  - Valor restante
  - Prazo
- Acompanhamento visual simples e motivador.

## 4. Relatórios Simples e Personalizados
- Área dedicada a relatórios que apresenta:
  - Total de receitas e despesas
  - Gastos por categoria (com gráficos visuais)
  - Gastos diários
- Interface clara e acessível, ideal para iniciantes.

## 5. Agente Financeiro
- Assistente conversacional que:
  - Registra gastos
  - Oferece **dicas de economia** personalizadas
- Torna a experiência mais prática e educativa.

## 6. Design Universal
- Interface inclusiva e acessível para diferentes perfis de usuários.
- Exemplos de práticas:
  - Linguagem clara
  - Contraste adequado
  - Suporte a leitores de tela
  - Navegação intuitiva
 
## Reflexão

### O que funcionou bem?
O refinamento do PRD previamento no Copilot ajudou, pois os créditos do Lovable terminaram em apenas 3 interações.

### O que não funcionou como o esperado?
Esperava poder interagir mais vezes gratuitamente com o Lovable, mas as interações feitas já foram muito úteis para aprender mais sobre Vibe Coding.

### O que aprendeu sobre conversar com IAs?
Aprendi que é basicamente igual a conversar com uma pessoa. Quanto mais detalhes e clareza se fornece, melhor é a interação.
